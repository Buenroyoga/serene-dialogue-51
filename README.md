# Diálogo Socrático Interior

Aplicación web para guiar un ritual socrático que transforma creencias nucleares con un flujo de perfil ACT, diagnóstico y diálogo en seis fases.

## Tecnologías

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Requisitos

- Node.js (recomendado con nvm)
- npm

## Configuración local

```sh
npm install
npm run dev
```

La app queda disponible en `http://localhost:8080`.

## Scripts útiles

- `npm run dev`: servidor de desarrollo
- `npm run build`: build de producción
- `npm run preview`: previsualización del build
- `npm run lint`: linting
- `npm run test`: pruebas unitarias
